# To check the node version
```
node --version
```

# To initialize the firebase console

```
firebase init
```

# To start the server 

```
npm run backend
```

# To create react project using npm

```
npm create-react-app <folder-name>   for ex--frontend
```

# To start the frontend or client 

```
npm start
```
